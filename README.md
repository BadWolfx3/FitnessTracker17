# Fitness Tracker
1/18/2021
Kelsey Busser

## Description
This is a fitness tracker that lets you view and create your own workouts. It lets you log multiple exercises in a workout on a given day. 

## Technologies Used
* HTML
* CSS
* JavaScript
* Mongoose
* Express
